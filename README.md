## 🌟 Overview

ResumeIQ is a state-of-the-art AI-powered resume optimization platform designed to help job seekers bridge the gap between their experience and modern Applicant Tracking Systems (ATS). By leveraging the reasoning power of Gemini 2.0 Flash, ResumeIQ provides deep diagnostic reviews and instant resume transformations.

## ✨ Key Features

- **🧠 Deep AI Analysis:** Instant diagnostic scoring of your resume based on impact, clarity, and technical depth.
- **🎯 ATS Optimization:** Identify missing keywords and get specific suggestions to pass automated filters.
- **✨ AI Resume Recreation:** Instantly rewrite your resume using different professional templates (Modern, Creative, Academic, Minimal) tailored to a specific job description.
- **📜 Version History:** Track your resume's improvement over time with a full archive of previous versions.
- **📄 Smart PDF Parsing:** Robust extraction of data from complex PDF layouts using customized parsing engines.

## 🛠️ Technology Stack

- **Frontend:** React 18, Vite, Tailwind CSS, Framer Motion (for premium animations)
- **Backend:** Node.js (ESM), Express
- **AI Engine:** Google Gemini 2.0 Flash & 1.5 Pro via `@google/genai` SDK
- **Database/Auth:** Firebase Authentication & Cloud Firestore
- **Deployment:** Google Cloud Run (Containerized with Docker)
- **Developed With:** Google Antigravity (AI Coding Assistant) & Google AI Studio

## 🛠️ Local Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/jayanitrivedi07-bit/ResumeIQ.git
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   cd frontend && npm install
   ```

3. **Set Environment Variables:**
   Create a `.env` file with your `GEMINI_API_KEY` and Firebase configuration.

4. **Run Locally:**
   ```bash
   npm run dev
   ```
